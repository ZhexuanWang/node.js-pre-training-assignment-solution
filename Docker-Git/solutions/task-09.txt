## Key Docker Commands Used

Image Management:
docker pull nginx:alpine          # Download image
docker tag nginx:alpine my-nginx  # Create custom tag
docker save/load                  # Backup/restore images

Container Management:
docker run -d -p 8080:80 nginx    # Detached + port mapping
docker exec -it container sh      # Shell access
docker cp host.txt container:/    # File transfer

Monitoring:
docker stats                      # CPU/memory usage
docker logs -f --timestamps       # Live logs
docker inspect container          # Full config

Cleanup:
docker system prune -a --volumes  # Remove everything unused
